# ذكاء محلي لهواتف Android 8+

تطبيق محادثة يعمل محليًا على الهاتف باستخدام MediaPipe LLM Inference.

## الاستخدام
1. ثبّت ملف APK.
2. نزّل نموذجًا متوافقًا بصيغة `.task` إلى الهاتف.
3. افتح التطبيق واضغط **اختيار نموذج**.
4. انتظر تشغيل النموذج، ثم ابدأ المحادثة دون إنترنت.
5. لتحليل صورة، استخدم نموذجًا يدعم الرؤية مثل Gemma 3n ثم اضغط **إضافة صورة**.

## ملاحظات مهمة
- الحد الأدنى Android 8.0 (API 26).
- نجاح تشغيل النموذج يعتمد على ذاكرة الهاتف والمعالج.
- النماذج الكبيرة قد لا تعمل على الأجهزة القديمة حتى لو تم تثبيت التطبيق.
- لا يُضمَّن النموذج داخل APK بسبب حجمه الكبير.
